# Datascience-Data_cleaning-1
 This is my First Data Science project for Data Cleaning. To clean the customer's call data set I have applied several approach in data set
 Some of the Import things I learned from this project are as follows:

 1. Removing Uncessary column in data set which are not useful for visualization.
 2. Replacing uncesary combination of character in Phone_number field.
 3. Also replacing unsual values such as nan--,  Na to space.
 4. Making usual i.e consistent data by replacing Yes, to Y and No to N only in column Do _Not_Contact.
 5. Using import function such as Loc to select data using row labels and column Name i.e important for removing uncessary row, but this can also be done using dropna.
 6. Finally Clean data is exported to Local machine.
